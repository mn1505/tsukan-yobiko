# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験をアプリ内で体系的に学べる個人予備校アプリを目指す、スマホファーストの静的Webアプリです。現在のバージョンはv1.1です。

v1.1は「関税法等カリキュラム拡充版」です。メモは補助であり、主役はアプリ内講義、確認問題、解説、ひっかけ解説、復習、分析です。

## このアプリの位置づけ

- 通関士試験学習の中心に置くスマホファースト個人予備校アプリ
- ヒュー赤は辞書、その他問題集はサブ演習として使う前提
- 市販教材や過去問本文の大量複製ではなく、オリジナル要約講義と自作確認問題で論点を整理
- 手入力メモより、理解度、正誤、復習対象、弱点タグ、今日のメニューを優先
- GitHub Pagesで公開できるHTML/CSS/JavaScriptのみの静的アプリ

## GitHub Pagesでの利用前提

このリポジトリはPublicで公開される前提です。コードと初期カリキュラムは公開されますが、学習記録、回答、メモ、弱点タグ、到達判定、今日のメニュー履歴、レッスン進捗などの個人データは各ブラウザのlocalStorageに保存され、GitHubには自動保存されません。

ローカルで使う場合は、リポジトリ直下の`index.html`をブラウザで開きます。GitHub Pagesで公開する場合は、GitHubのリポジトリ設定からPagesを有効にし、公開元をリポジトリのルートに設定します。React、Vue、外部ライブラリ、CDN、外部API通信は使っていません。

## 保存方式

保存先はブラウザのlocalStorageです。

- 単元データ: `tsukanYobiko.units`
- バージョン: `tsukanYobiko.version`
- 演習ログ用: `tsukanYobiko.practiceLogs`
- 過去問ログ用: `tsukanYobiko.pastExamLogs`
- 実務ログ用: `tsukanYobiko.practicalLogs`
- AI解析ログ用: `tsukanYobiko.aiAnalyses`
- 今日の学習メニュー用: `tsukanYobiko.studyPlans`
- レッスン進捗用: `tsukanYobiko.curriculumProgress`

既存の`curriculumProgress`は壊さず、レッスン本体はアプリ内初期データとして持ちます。新規レッスンが追加されても、既存レッスンの進捗は保持されます。

バックアップJSONには、`units`、`practiceLogs`、`pastExamLogs`、`practicalLogs`、`aiAnalyses`、`studyPlans`、`curriculumProgress`が含まれます。JSON復元と初期データリセットは別機能です。

## 今日のメニュー生成

今日のメニューには、関税法等カリキュラムの進捗が反映されます。

- C判定の関税法等レッスン
- 復習対象の関税法等レッスン
- 未着手の次レッスン
- 関税法等ミニ模試B判定以下
- 保税・納税・課税価格など頻出論点の弱点レッスン
- 最終学習日が古いレッスン

各メニュー項目には、種別、理由、優先度、推定時間、関連単元・ログ・レッスンが表示されます。

## AI機能について

v1.1ではAI API連携は未実装です。外部通信は行わず、生成したプロンプトをChatGPTなどに手動で貼り付けて使います。

AIプロンプト生成では、対象データに「関税法等カリキュラム」を追加しました。関税法等レッスン進捗、A/B/C判定、間違えた確認問題、弱点タグ、ミニ模試結果、復習対象レッスン、保税・納税・課税価格・減免税などの苦手状況を含めて、総合学習相談や弱点抽出に使えます。

## Publicリポジトリ運用時の注意

- コードと初期カリキュラムは公開されます
- 個人の学習記録はlocalStorageに保存され、GitHubには自動保存されません
- スマホとPCは自動同期されません
- 定期的にJSONバックアップを取ってください
- 個人情報、秘密情報、非公開情報を初期データとしてコードに書き込まないでください
- 市販教材や過去問本文をそのまま大量に初期データへ入れないでください
- 法改正があるため、最新法令は別途確認してください
