# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験学習の8割を担うことを目指すスマホファーストの個人予備校アプリです。現在のバージョンはv0.8です。v0.8は暫定完成版として、単元管理、演習ログ、過去問分析、実務ログ、AI解析プロンプト生成、弱点分析、復習管理、今日の学習メニュー、JSONバックアップ/復元をlocalStorageベースで扱えます。

## このアプリの位置づけ

- 通関士試験学習の8割を担うスマホファースト個人予備校アプリ
- 合格レベルは、過去問を安定して解答できる状態
- 市販教材や過去問本文を大量複製する用途ではなく、法令・制度・試験論点・自分のミスを整理するためのアプリ
- 今日のメニューは学習補助の目安であり、必要に応じてユーザーが調整してください

## GitHub Pagesでの利用前提

このリポジトリはPublicで公開される前提です。コードと初期データは公開されますが、学習記録、回答、メモ、弱点タグ、到達判定、今日のメニュー履歴などの個人データは各ブラウザのlocalStorageに保存され、GitHubには自動保存されません。

ローカルで使う場合は、リポジトリ直下の`index.html`をブラウザで開きます。GitHub Pagesで公開する場合は、GitHubのリポジトリ設定からPagesを有効にし、公開元をリポジトリのルートに設定します。React、Vue、外部ライブラリ、CDN、外部API通信は使っていません。

## 保存方式

保存先はブラウザのlocalStorageです。

- 単元データ: `tsukanYobiko.units`
- バージョン: `tsukanYobiko.version`
- 演習ログ用: `tsukanYobiko.practiceLogs`
- 過去問ログ用: `tsukanYobiko.pastExamLogs`
- 実務ログ用: `tsukanYobiko.practicalLogs`
- AI解析ログ用: `tsukanYobiko.aiAnalyses`
- 今日の学習メニュー用: `tsukanYobiko.studyPlans`

スマホで入力したデータはスマホブラウザに、PCで入力したデータはPCブラウザに保存されます。スマホとPCは自動同期されません。定期的にJSONバックアップを取ってください。

バックアップJSONには、`units`、`practiceLogs`、`pastExamLogs`、`practicalLogs`、`aiAnalyses`、`studyPlans`が含まれます。古いバックアップに`studyPlans`が含まれていない場合も、空配列として扱いエラーにしません。

## 今日のメニュー生成

優先順位の主な考え方は以下です。

- 危険度スコアが高い単元
- 最優先復習の単元
- 到達判定C
- 関連過去問ログ、実務ログ、演習ログに×がある単元
- 再演習対象
- 弱点タグが多い単元
- 最終更新日が古い単元
- 重要度が高い単元

各メニュー項目には、種別、理由、優先度、推定時間、関連単元・ログが表示されます。完了チェックと今日のメモは当日の`studyPlans`に保存され、再読み込み後も同じ日の状態が残ります。

## 危険度・復習判定

危険度は保存データではありません。現在の`units`、`practiceLogs`、`pastExamLogs`、`practicalLogs`、`aiAnalyses`から画面表示時に自動計算されます。

危険度スコアの主な計算ルール:

- 到達判定Cは+30、Bは+15
- 復習対象、再演習対象はそれぞれ+10
- 単元の弱点タグ1件につき+5
- 例題や単元詳細の過去問で×は+15、△は+8
- 関連演習ログの×は+10、△は+5、再演習対象は+5
- 関連過去問ログの×は+15、△は+8、再演習対象は+5、優先度「高」は+10
- 関連実務ログの×は+15、△は+8、再演習対象は+5、優先度「高」は+10、実務用弱点タグ1件につき+5

危険度の目安は、0〜9が安定、10〜24が注意、25〜49が危険、50以上が最優先改善です。分析結果は学習優先順位を決めるための目安であり、最終判断はユーザー自身が行ってください。

## AI機能について

v0.8ではAI API連携は未実装です。外部通信は行わず、生成したプロンプトをChatGPTなどに手動で貼り付けて使います。「今日のメニューをAIに相談」では、プロンプト種別を総合学習相談、対象データを全体サマリーに設定し、今日のメニューと完了状況もプロンプトの全体サマリーに含めます。

## Publicリポジトリ運用時の注意

- コードと初期データは公開されます
- 個人の学習記録はlocalStorageに保存され、GitHubには自動保存されません
- スマホとPCは自動同期されません
- 定期的にJSONバックアップを取ってください
- バックアップファイルには学習記録、演習ログ、過去問ログ、実務ログ、AIプロンプト履歴、今日のメニュー、AI返答メモ、回答メモ、ミス理由、弱点タグが含まれるため、取り扱いに注意してください
- 個人情報、秘密情報、非公開情報を初期データとしてコードに書き込まないでください
- 市販教材や過去問本文をそのまま大量に初期データへ入れないでください
