# TSUKAN YOBIKO

現在のバージョン: v3.1

TSUKAN YOBIKOは、通関士試験の独学者向けローカル教材アプリです。講義、確認問題、ドリル、弱点別復習、模試、過去問マッピング、JSONバックアップをブラウザ内で管理します。

## v3.1の位置づけ

v3.1は「過去問10年分分析に基づく教材補強基盤版」です。

過去問本文をそのまま大量登録するのではなく、過去問分析から抽出した論点・弱点・ひっかけパターンを、オリジナル講義・オリジナル問題・ドリル・模試へ変換して補強します。

## 教材補強方針

- AI APIは使いません。
- 外部通信は行いません。
- OpenAI APIキーは不要です。
- Cloudflare Workers連携はありません。
- 過去問本文をそのまま大量収録しません。
- 市販教材や試験問題の解説をコピーしません。
- 過去問分析から、論点・弱点・ひっかけパターンとして教材化します。
- 追加問題はオリジナル問題として作成します。

## 過去問10年分分析の運用メモ

1. 過去問PDFをChatGPTにアップロードする
2. 年度別・科目別・問題別に分析する
3. 頻出論点を抽出する
4. 不足教材を洗い出す
5. 教材補強パックを作る
6. Codexで `data/lessons.js` と `data/question-bank.js` に反映する
7. GitHub Pagesで動作確認する
8. ドリル・模試に反映されているか確認する
9. JSONバックアップを取る

## AI APIについて

アプリ上ではAI APIを使用しません。外部通信は行いません。不明点や教材補強案は、外部ChatGPT相談文をコピーして、ユーザーがChatGPTに貼り付けて相談する運用です。

## データ保存

学習データはブラウザのlocalStorageに保存されます。ブラウザや端末を変えると自動では引き継がれません。ブラウザデータ削除、キャッシュ削除、端末変更などで消える可能性があります。

定期的にJSONバックアップを作成し、安全な場所に保存してください。

## 初回利用手順

1. GitHub Pagesで開く
2. 学習画面でレッスンを読む
3. 確認問題を解く
4. ドリルを解く
5. 復習画面で弱点を直す
6. 模試を受ける
7. 過去問JSONを取り込んで教材根拠率を確認する
8. 分析画面で教材補強パックと全範囲補強マップを確認する
9. バックアップを取る

## 過去問マッピングの使い方

1. 過去問JSONを用意する
2. アプリにインポートする
3. 各問題に対応レッスン・対応問題を紐づける
4. A/B/C/Dで教材根拠判定する
5. 教材根拠率を見る
6. 不足教材リストを確認する
7. 不足論点を教材補強パック化する

判定の目安は、Aが教材だけで十分対応可能、Bが一部補足で対応可能、Cが追加教材が必要、Dが未対応です。

## バックアップ手順

1. 設定画面を開く
2. JSONバックアップを作成
3. ファイルを安全な場所へ保存
4. 復元時は復元プレビューを確認する
5. 「現在のデータをバックアップ済み」と「上書きを理解しました」を確認してから復元する

バックアップには、学習進捗、ドリル結果、模試結果、lessonOverrides、インポート済み過去問、過去問マッピングなどが含まれます。

## 注意事項

TSUKAN YOBIKOはローカル教材アプリです。教材データと学習データはブラウザ内で扱い、アプリから外部通信は行いません。過去問データはユーザーがJSONとして取り込み、教材化するときは本文の丸写しではなく、論点・弱点・ひっかけパターンとしてオリジナル教材に変換してください。
