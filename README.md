# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験学習の8割を担うことを目指すスマホファーストの個人予備校アプリです。現在のバージョンはv0.9です。

v0.9では、方針を「ログ管理中心」から「学習エンジン中心」に変更しました。学習内容を自分で大量にメモするためのアプリではなく、アプリ内で講義を読み、確認問題を解き、解説でひっかけを理解し、復習へ回すための予備校アプリを目指します。メモ機能は補助であり、主役は講義・確認問題・解説・復習です。

## このアプリの位置づけ

- 通関士試験学習の中心に置くスマホファースト個人予備校アプリ
- 合格レベルは、過去問を安定して解答できる状態
- 市販教材や過去問本文を大量複製する用途ではなく、オリジナルの要約講義と自作確認問題で論点を整理するアプリ
- 手入力メモは補助扱いで、学習完了、正誤、理解度、復習対象、弱点タグを優先して記録
- 今日のメニューは学習補助の目安であり、必要に応じてユーザーが調整

## GitHub Pagesでの利用前提

このリポジトリはPublicで公開される前提です。コードと初期カリキュラムは公開されますが、学習記録、回答、メモ、弱点タグ、到達判定、今日のメニュー履歴、レッスン進捗などの個人データは各ブラウザのlocalStorageに保存され、GitHubには自動保存されません。

ローカルで使う場合は、リポジトリ直下の`index.html`をブラウザで開きます。GitHub Pagesで公開する場合は、GitHubのリポジトリ設定からPagesを有効にし、公開元をリポジトリのルートに設定します。React、Vue、外部ライブラリ、CDN、外部API通信は使っていません。

## 保存方式

保存先はブラウザのlocalStorageです。

- 単元データ: `tsukanYobiko.units`
- バージョン: `tsukanYobiko.version`
- 演習ログ用: `tsukanYobiko.practiceLogs`
- 過去問ログ用: `tsukanYobiko.pastExamLogs`
- 実務ログ用: `tsukanYobiko.practicalLogs`
- AI解析ログ用: `tsukanYobiko.aiAnalyses`
- 今日の学習メニュー用: `tsukanYobiko.studyPlans`
- レッスン進捗用: `tsukanYobiko.curriculumProgress`

スマホで入力したデータはスマホブラウザに、PCで入力したデータはPCブラウザに保存されます。スマホとPCは自動同期されません。定期的にJSONバックアップを取ってください。

バックアップJSONには、`units`、`practiceLogs`、`pastExamLogs`、`practicalLogs`、`aiAnalyses`、`studyPlans`、`curriculumProgress`が含まれます。古いバックアップに`studyPlans`や`curriculumProgress`が含まれていない場合も、空配列として扱いエラーにしません。カリキュラム本体はアプリ内の初期データとして持ち、バックアップ対象は進捗データのみです。

## 今日のメニュー生成

今日のメニューには、従来の単元・演習・過去問・実務ログに加えて、レッスン学習とレッスン復習が含まれます。

優先順位の主な考え方は以下です。

- C判定のレッスン
- `reviewNeeded`がtrueのレッスン
- 未着手の次レッスン
- 最終学習日が古いレッスン
- 関連単元の危険度が高いレッスン
- 罰則・処分・手続など混同しやすいレッスン
- 危険度スコアが高い単元
- 関連過去問ログ、実務ログ、演習ログに×がある単元
- 再演習対象
- 弱点タグが多い単元

各メニュー項目には、種別、理由、優先度、推定時間、関連単元・ログ・レッスンが表示されます。完了チェックと今日のメモは当日の`studyPlans`に保存され、再読み込み後も同じ日の状態が残ります。

## AI機能について

v0.9ではAI API連携は未実装です。外部通信は行わず、生成したプロンプトをChatGPTなどに手動で貼り付けて使います。

AIプロンプト生成では、対象データに「レッスン」を追加しました。レッスンタイトル、科目、学習目標、講義本文の要約、重要ポイント、混同ポイント、引っかけ注意、試験で狙われる表現、確認問題の結果、理解度判定、復習対象かどうかを含めてプロンプトを生成できます。

## Publicリポジトリ運用時の注意

- コードと初期カリキュラムは公開されます
- 個人の学習記録はlocalStorageに保存され、GitHubには自動保存されません
- スマホとPCは自動同期されません
- 定期的にJSONバックアップを取ってください
- バックアップファイルには学習記録、演習ログ、過去問ログ、実務ログ、AIプロンプト履歴、今日のメニュー、AI返答メモ、回答メモ、ミス理由、弱点タグ、レッスン進捗が含まれるため、取り扱いに注意してください
- 個人情報、秘密情報、非公開情報を初期データとしてコードに書き込まないでください
- 市販教材や過去問本文をそのまま大量に初期データへ入れないでください
