# TSUKAN YOBIKO

通関士試験向けのスマホファースト個人予備校アプリです。GitHub Pagesで動く静的アプリとして、HTML / CSS / JavaScriptのみで構成しています。

## 現在のバージョン

v2.5

v2.5は「総合模試セット追加版」です。ローカルのオリジナル問題バンクから、通関業法・関税法等・通関実務を横断した模試を複数セット出題できるようにしています。

## v2.5の大方針

- アプリ上でAI APIは使いません
- OpenAI API連携は復活させません
- Cloudflare Workers連携は使いません
- 外部通信は行いません
- 不明点や深掘りは外部ChatGPT相談文をコピーして行います
- アプリ本体はローカル教材・問題・解説・復習・模試・ドリル結果分析に集中します
- 模試問題はローカルのオリジナル問題バンクから出題されます
- 問題バンクの問題は、市販教材や過去問本文の丸写しではないオリジナル問題です
- 既存の過去AIメモや `lessonOverrides` 相当の追加教材は、学習データとして読める形を維持します
- v2.5で教材セット投入フェーズを一旦区切り、以降はデータ分離とUI整理へ進みます

## 中心機能

- 学ぶ：講義、重要ポイント、混同ポイント、ひっかけ注意、ミニまとめ
- 解く：確認問題、弱点別ドリル、弱点グループ別ドリル、論点別ドリル、科目別ミニ模試、総合模試
- 直す：解説、ひっかけ解説、ミス防止ポイント、復習対象化
- 繰り返す：C判定復習、誤答復習、弱点タグ別復習、今日のメニュー
- 見る：進捗、弱点、模試結果、科目別状態、問題バンク分析、弱点タグ別正答率

## データ保存

学習データはこのブラウザの `localStorage` に保存されます。スマホとPCは自動同期されません。定期的にJSONバックアップを取ってください。

エクスポートJSONの主な保存対象:

- `units`
- `practiceLogs`
- `pastExamLogs`
- `practicalLogs`
- `aiAnalyses`
- `studyPlans`
- `curriculumProgress`
- `mockExamResults`
- `lessonOverrides`
- `drillResults`

古いバックアップに `mockExamResults` や `drillResults` が含まれていなくても、復元時にエラーにならないようにしています。JSON復元と初期データリセットは別機能です。

## 外部ChatGPT相談

v2.5ではアプリ内通信を行いません。必要な場合は、アプリで相談文を作成し、外部ChatGPTに貼り付けて使います。

模試結果・弱点別ドリル結果を含めた相談例:

- 総合模試の科目別正答率と弱点タグから、次に復習すべき順番を提案してください
- ひっかけ総点検模試の誤答から、選択肢読解と主体・権限者の混同を整理してください
- 弱点別ドリルの結果をもとに、復習順を提案してください
- 課税価格タグの誤答が多いので、加算要素と不算入要素を整理してください
- 罰則・処分グループの正答率が低いので、通関業法と関税法等を横断して整理してください
- 品目分類・資料読取グループの弱点を分析してください

## 不要になったファイル

v2.0以降、Cloudflare Workers用のAI中継サーバー関連ファイルは不要です。

- `worker.js`
- `wrangler.toml.example`
- `workers/` 配下のAI中継サーバー関連ファイル
