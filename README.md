# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験学習の8割を担うことを目指すスマホファーストの個人予備校アプリです。v0.1では、単元整理、演習メモ、過去問対応メモ、弱点タグ、到達判定、復習対象管理をlocalStorageに保存できる土台を実装しています。

## v0.1の目的

- GitHub Pagesでそのまま動く静的アプリにする
- スマホブラウザで毎日記録しやすい基本UIを作る
- 通関士試験の単元、法令・制度、試験論点、演習結果、復習状況を蓄積できるようにする
- 将来のAI解析、過去問分析、JSONバックアップ/復元に拡張しやすいデータ構造を用意する

## このアプリの位置づけ

- 通関士試験学習の8割を担うスマホファースト個人予備校アプリ
- ヒューマンアカデミー赤テキストは辞書、その他問題集はサブ演習として使う
- 合格レベルは第59回通関士試験の過去問を安定して解ける状態
- 市販教材の本文を大量複製する用途ではなく、法令・制度・試験論点を自分用に整理するためのアプリ

## GitHub Pagesでの利用前提

このリポジトリはPublicで公開される前提です。コードと初期データは公開されますが、学習記録、回答、メモ、弱点タグ、到達判定などの個人データは各ブラウザのlocalStorageに保存され、GitHubには自動保存されません。

## 起動方法

ローカルで使う場合は、リポジトリ直下の`index.html`をブラウザで開きます。

GitHub Pagesで公開する場合は、GitHubのリポジトリ設定からPagesを有効にし、公開元をリポジトリのルートに設定します。`index.html`、`style.css`、`script.js`は相対パスで読み込まれるため、Pages上のURL配下でも動作します。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

React、Vue、外部ライブラリ、CDN、外部API通信はv0.1では使っていません。

## 保存方式

保存先はブラウザのlocalStorageです。

- 単元データ: `tsukanYobiko.units`
- バージョン: `tsukanYobiko.version`
- 将来用: `tsukanYobiko.practiceLogs`
- 将来用: `tsukanYobiko.pastExamLogs`
- 将来用: `tsukanYobiko.aiAnalyses`

スマホで入力したデータはスマホブラウザに、PCで入力したデータはPCブラウザに保存されます。v0.1ではスマホとPCの自動同期はありません。

## Publicリポジトリ運用時の注意

- コードと初期データは公開されます
- 個人の学習記録はlocalStorageに保存され、GitHubには自動保存されません
- 個人情報、秘密情報、非公開情報を初期データとしてコードに書き込まないでください
- 市販教材の本文をそのまま大量に初期データへ入れないでください

## 到達判定

- A: 本試験で出ても正答できるレベル
- B: 理解はあるが、選択肢で迷う可能性あり
- C: 制度趣旨・要件・例外の整理が不十分
- 未判定: まだ確認していない

## 復習判定のルール

最優先復習:

- 到達判定がC
- 例題の採点結果に×がある
- 過去問正誤が×

通常復習:

- 到達判定がB
- 例題の採点結果に△がある
- 復習対象がtrue
- 過去問正誤が△

復習不要:

- 上記に該当しない単元

同じ単元が複数条件に該当する場合は、最も重い判定を優先します。
