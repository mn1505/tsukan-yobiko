# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験の独学者向けローカル教材アプリです。講義、確認問題、ドリル、弱点別復習、模試、過去問マッピングをブラウザ内で管理します。

## v3.0.1 修正内容

v3.0.1では、ドリル画面の開始ボタンを共通の開始処理に接続し直し、通関業法・関税法等・通関実務・弱点別・手順・計算過程・資料読取・ひっかけ総点検の各ドリルがQUESTION_BANKから問題を抽出して開始できるように安定化しました。対象問題が0件の場合は画面内メッセージとconsole.warnで読み込み状態を確認できるようにしています。

## v3.0の位置づけ

v3.0は「ローカル予備校版完成」です。GitHub Pagesで公開できる静的アプリとして、毎日の学習、復習、模試、過去問マッピング、JSONバックアップ運用までを一旦完成版として整理しています。

## AI APIについて

- アプリ上ではAI APIを使用しません。
- 外部通信は行いません。
- OpenAI APIキーは不要です。
- 不明点は、外部ChatGPT相談文をコピーして、ChatGPTに貼り付けて相談する運用です。

## データ保存

学習データはブラウザのlocalStorageに保存されます。ブラウザや端末を変えると自動では引き継がれません。ブラウザデータ削除、キャッシュ削除、端末変更などで消える可能性があります。

定期的にJSONバックアップを作成し、安全な場所に保存してください。

## 初回利用手順

1. GitHub Pagesで開く
2. 学習画面でレッスンを読む
3. 確認問題を解く
4. ドリルを解く
5. 復習画面で弱点を直す
6. 模試を受ける
7. 過去問JSONを取り込んで教材根拠率を確認する
8. バックアップを取る

## 過去問マッピングの使い方

1. 過去問JSONを用意する
2. アプリにインポートする
3. 各問題に対応レッスン・対応問題を紐づける
4. A/B/C/Dで教材根拠判定する
5. 教材根拠率を見る
6. 不足教材リストを確認する

判定の目安は、Aが教材だけで十分対応可能、Bが一部補足で対応可能、Cが追加教材が必要、Dが未対応です。

## 過去問JSON形式

`sample-past-exam.json`を参考に、次のような配列またはオブジェクトを用意します。

```json
{
  "questions": [
    {
      "id": "past-2025-tsukangyoho-01",
      "examName": "通関士試験",
      "year": "2025",
      "round": "第59回",
      "subject": "通関業法",
      "questionNo": "第1問",
      "topic": "通関業者の義務",
      "questionText": "問題文または要約を入力します。",
      "answer": "正答メモ",
      "explanation": "解説メモ",
      "weaknessTag": "通関業者の義務"
    }
  ]
}
```

## バックアップ手順

1. 設定画面を開く
2. JSONバックアップを作成
3. ファイルを安全な場所へ保存
4. 復元時は復元プレビューを確認する
5. 「現在のデータをバックアップ済み」と「上書きを理解しました」を確認してから復元する

バックアップには、学習進捗、ドリル結果、模試結果、lessonOverrides、インポート済み過去問、過去問マッピングなどが含まれます。
