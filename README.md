# TSUKAN YOBIKO

通関士試験向けのスマホファースト個人予備校アプリです。GitHub Pagesで動く静的アプリとして、HTML / CSS / JavaScriptのみで構成しています。

## 現在のバージョン

v2.2

v2.2は「関税法等 問題バンク200問追加版」です。保税、輸出入申告、納税、課税価格、減免税、罰則をローカル問題バンクで反復できるようにしています。

## v2.2の大方針

- アプリ上でAI APIは使いません
- OpenAI API連携は復活させません
- Cloudflare Workers連携は使いません
- 外部通信は行いません
- 不明点や深掘りは外部ChatGPT相談文をコピーして行います
- アプリ本体はローカル教材・問題・解説・復習・模試・ドリル結果分析に集中します
- 問題バンクの問題は、市販教材や過去問本文の丸写しではないオリジナル問題です
- 既存の過去AIメモや `lessonOverrides` 相当の追加教材は、学習データとして読める形を維持します

## 中心機能

- 学ぶ：講義、重要ポイント、混同ポイント、ひっかけ注意、ミニまとめ
- 解く：確認問題、弱点別ドリル、論点別ドリル、科目別ミニ模試、総合模試
- 直す：解説、ひっかけ解説、ミス防止ポイント、復習対象化
- 繰り返す：C判定復習、誤答復習、弱点タグ別復習、今日のメニュー
- 見る：進捗、弱点、模試結果、科目別状態、問題バンク分析

## データ保存

学習データはこのブラウザの `localStorage` に保存されます。スマホとPCは自動同期されません。定期的にJSONバックアップを取ってください。

エクスポートJSONの主な保存対象:

- `units`
- `practiceLogs`
- `pastExamLogs`
- `practicalLogs`
- `aiAnalyses`
- `studyPlans`
- `curriculumProgress`
- `mockExamResults`
- `lessonOverrides`
- `drillResults`

古いバックアップに `drillResults` が含まれていなくても、復元時にエラーにならないようにしています。JSON復元と初期データリセットは別機能です。

## 外部ChatGPT相談

v2.2ではアプリ内通信を行いません。必要な場合は、アプリで相談文を作成し、外部ChatGPTに貼り付けて使います。

作成できる相談文:

- レッスンについての質問文
- 誤答についての質問文
- 模試結果についての質問文
- 弱点タグについての質問文
- 今日のメニューについての質問文
- 通関業法ドリル結果についての誤答分析相談文
- 関税法等ドリル結果についての誤答分析相談文
- 保税地域と保税運送の混同整理
- 納期限・法定納期限、延滞税・加算税の関係整理
- 課税価格の加算要素と不算入要素の復習

## 不要になったファイル

v2.0以降、Cloudflare Workers用のAI中継サーバー関連ファイルは不要です。

- `worker.js`
- `wrangler.toml.example`
- `workers/` 配下のAI中継サーバー関連ファイル
