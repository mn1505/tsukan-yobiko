# TSUKAN YOBIKO

TSUKAN YOBIKOは、通関士試験をアプリ内で体系的に学べる個人予備校アプリを目指す、スマホファーストの静的Webアプリです。現在のバージョンはv1.3です。

v1.3は「総合模試・横断復習版」です。科目別学習に加えて、通関業法・関税法等・通関実務を横断する本試験型の演習、採点、解説、復習、分析を強化しました。

## このアプリの位置づけ

- 通関士試験学習の中心に置くスマホファースト個人予備校アプリ
- ヒュー赤は辞書、その他問題集はサブ演習として使う前提
- 市販教材や過去問本文の大量複製ではなく、オリジナル要約講義と自作確認問題で論点を整理
- 手入力メモより、講義、確認問題、模試、解説、理解度、復習対象、弱点タグ、今日のメニューを優先
- GitHub Pagesで公開できるHTML/CSS/JavaScriptのみの静的アプリ

## 総合模試・横断復習

v1.3では、独立した「模試」画面を追加しました。

- 15問ライト模試: 通関業法5問、関税法等5問、通関実務5問
- 30問標準模試: 通関業法10問、関税法等12問、通関実務8問
- 弱点集中模試: C判定レッスン、復習対象、模試誤答、弱点タグ上位を優先

採点後は、総問題数、正解数、正答率、A/B/C判定、科目別正答率、弱点タグ、間違えた問題、通常解説、ひっかけ解説、関連レッスンを確認できます。C判定の模試は復習対象として扱われます。

横断復習モードでは、C判定レッスン、復習対象レッスン、模試で間違えた問題、演習ログ・過去問ログ・実務ログの×や△、弱点タグ上位を科目横断で表示します。

## GitHub Pagesでの利用前提

このリポジトリはPublicで公開される前提です。コードと初期カリキュラム、オリジナル模試問題は公開されますが、学習記録、模試結果、回答、メモ、弱点タグ、到達判定、今日のメニュー履歴、レッスン進捗などの個人データは各ブラウザのlocalStorageに保存され、GitHubには自動保存されません。

ローカルで使う場合は、リポジトリ直下の`index.html`をブラウザで開きます。GitHub Pagesで公開する場合は、GitHubのリポジトリ設定からPagesを有効にし、公開元をリポジトリのルートに設定します。React、Vue、外部ライブラリ、CDN、外部API通信は使っていません。

## 保存方式

保存先はブラウザのlocalStorageです。

- 単元データ: `tsukanYobiko.units`
- バージョン: `tsukanYobiko.version`
- 演習ログ用: `tsukanYobiko.practiceLogs`
- 過去問ログ用: `tsukanYobiko.pastExamLogs`
- 実務ログ用: `tsukanYobiko.practicalLogs`
- AI解析ログ用: `tsukanYobiko.aiAnalyses`
- 今日の学習メニュー用: `tsukanYobiko.studyPlans`
- レッスン進捗用: `tsukanYobiko.curriculumProgress`
- 総合模試結果用: `tsukanYobiko.mockExamResults`

既存の`curriculumProgress`は壊さず、レッスン本体と模試問題はアプリ内初期データとして持ちます。新規機能が追加されても、既存レッスンの進捗は保持されます。

バックアップJSONには、`units`、`practiceLogs`、`pastExamLogs`、`practicalLogs`、`aiAnalyses`、`studyPlans`、`curriculumProgress`、`mockExamResults`が含まれます。古いバックアップに`mockExamResults`がない場合は空配列として扱います。JSON復元と初期データリセットは別機能です。

## 今日のメニュー生成

今日のメニューには、総合模試と横断復習が反映されます。

- 前回模試C判定
- 前回模試で間違えた論点
- 弱点タグ上位
- C判定レッスン
- 実務・過去問・演習の×ログ
- 未実施の総合模試

15分なら横断復習、30分なら横断復習または15問ライト模試、1時間ならライト模試と誤答復習、2時間以上なら30問標準模試と弱点復習を優先します。

## AI機能について

v1.3でもAI API連携は未実装です。外部通信は行わず、生成したプロンプトをChatGPTなどに手動で貼り付けて使います。

AIプロンプト生成では、対象データに「総合模試結果」「最新模試結果」「横断弱点」を追加しました。模試モード、正答率、A/B/C判定、科目別正答率、間違えた問題、弱点タグ、関連レッスン、次に復習すべき候補を含めて、総合学習相談、弱点抽出、復習指示に使えます。

## Publicリポジトリ運用時の注意

- コードと初期カリキュラム、オリジナル模試問題は公開されます
- 個人の学習記録と模試結果はlocalStorageに保存され、GitHubには自動保存されません
- スマホとPCは自動同期されません
- 定期的にJSONバックアップを取ってください
- 個人情報、秘密情報、非公開情報を初期データとしてコードに書き込まないでください
- 市販教材や過去問本文をそのまま大量に初期データへ入れないでください
- 法改正があるため、最新法令は別途確認してください
